Make a scivision catalog entry

This is a mock-up/early prototype. Start with the Scivision GitHub repo.

Use the form below to create a datasource catalog entry. Alternatively, generate a model catalog entry here.

Steps

  • Complete the form below and click 'Download'. Save the resulting json file
  • Edit the Datasource Catalog on GitHub, adding the contents of the file
    • Direct link to edit the catalog - this will prompt you to sign in to GitHub, and will fork the scivision repository to your account (if you don't have write permission)
    • Click 'propose changes' on the GitHub editor to create a new branch and open a pull request
  • A maintainer will review the PR and suggest changes
    • After merging the PR, your entry has been included in the catalog - congratulations!
    • The maintainer should make a release so that this entry is available in the PyPI version of scivision

See the documentation on extending the catalog for further details.
